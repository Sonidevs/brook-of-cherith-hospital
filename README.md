# Brook of Cherith Hospital Website

Welcome to the **Brook of Cherith Hospital** website repository. This website is designed to provide information about the hospital, its departments, services, doctors, latest news, and resources for patients.

## 🌐 Live Website
You can view the live website here:  
[Brook of Cherith Hospital](https://yourusername.github.io/brook-of-cherith-hospital/)

## 🏥 Features
- Home page with hero section and hospital introduction
- About Us section (mission, vision, core values)
- Services offered by the hospital
- Departments and specialized clinics
- Resource Centre with patient guides and health education
- Doctors’ profiles
- Latest news and announcements
- Testimonials from patients
- Patient form (connected with Formspree)
- Contact information

## 📂 Files
- `index.html` – main homepage
- `style.css` – custom styles (embedded in HTML in this version)
- Images – hosted online (or you can store locally in `/images`)
- Form integrated with [Formspree](https://formspree.io/f/xpwyqrwd)
